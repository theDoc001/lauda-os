# Pending decisions

Agents write proposals here. The owner commits them by hand.

## Format

Each pending item is a second-level heading with a compact body.

```markdown
## {YYYY-MM-DD}: {one-line title} (from: {agent})

**Proposal**: {what the agent wants to do}
**Why**: {the reason, short}
**Impact**: {files / folders / principles affected}
**Status**: pending | accepted | rejected | superseded
**Resolved**: {YYYY-MM-DD if not pending}
```

Leave accepted and rejected items in place for the archive. Scrutineer counts only items with `Status: pending`.
